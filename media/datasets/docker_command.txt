To build application : sudo docker-compose build or docker build . -t visu-data

To Run server or app : sudo docker-compose up

To show all images : sudo docker ps -a

To stop process : sudo docker stop PID

To Enter in docker shell : sudo docker exec -it {PID} bash 

To exit outside : exit

stop all running containers: sudo docker stop $(sudo docker ps -aq)

remove all stopped containers: sudo docker rm $(sudo docker ps -aq)

To delete all Docker images: sudo docker rmi $(sudo docker images -aq)
